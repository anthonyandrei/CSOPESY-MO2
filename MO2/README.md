## How to Build and Run

### Prerequisites

- Windows
- Visual Studio 2022 with C++ Desktop development workload

### Build and Run

1. Clone GitHub repository
2. Open CSOPESY-MO2.sln
3. Click Local Windows Debugger in Visual Studio
